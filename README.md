# Lab8 Async Service (Django)

Асинхронный сервис (порт `8001`) для отложенного вычисления `result_stage` по среднему АД.

## Формула

Среднее АД (MAP):

$$MAP = \frac{SYS + 2\cdot DIA}{3}$$

Дальше выбирается «стадия»:
- `MAP < 100` → `Стадия 1`
- `100 ≤ MAP < 120` → `Стадия 2`
- `MAP ≥ 120` → `Стадия 3`
